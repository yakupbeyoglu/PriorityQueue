Detailed Description

template<typename T, typename Compare = std::less<T>>
class yb::priority_queue< T, Compare >

Priority queue class template.

This template represents a priority queue that can store elements of type T. The elements are sorted in descending order based on the specified comparison function.

Template Parameters
TThe type of elements to be stored in the priority queue.
CompareThe comparison function for ordering elements. Defaults to std::less<T>.
